# Электроника МС 0511 (УКНЦ)
## разработка на С
### GCC кросскомпиляция

Собираю здесь все что связано с разработкой на C, предпочтительно с использованием кросскомпиляции GCC

1. В папке `gcc` лежит скрипт `build_gcc_uknc.sh` с помощью котороко можно собрать тулчейн для сборки проектов для УКНЦ. 

2. В папке `example` лежит минимальный пример приложения собираемого с помощью тулчейна получаемого скриптом из пункта 1.
